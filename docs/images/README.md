# 截图素材目录

将项目运行截图放在此目录，README 会自动引用。

建议文件名：

| 文件 | 内容 |
|------|------|
| `home.png` | 首页：模块选择 + 考点 + 生成按钮 |
| `generate-loading.png` | 生成中 Loading + 取消按钮 |
| `questions.png` | 题目列表 + 来源标签 |
| `publish-xhs.png` | 小红书发布面板 |
| `publish-douyin.png` | 抖音发布面板 + 配图预览 |
| `ai-provider.png` | AI 提供商下拉（已配置/未配置状态） |
| `vocab.png` | 700 高频词面板 |

截图方式：浏览器访问 http://localhost:5173 ，按 `Cmd+Shift+4`（Mac）或 Win+Shift+S 截取后保存到此目录。
